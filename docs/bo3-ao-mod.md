# BO3 AO-Mod (Version 2.4b) [ZEN].gpc

[`BO3 AO-Mod (Version 2.4b) [ZEN].gpc`](../Zen%20Scripts/BO3%20AO-Mod%20%28Version%202.4b%29%20%5BZEN%5D.gpc) is a Cronus Zen script for Call of Duty: Black Ops 3. It includes gameplay toggles such as Rapid Fire, Anti-Recoil, Jitter, Drop Shot, Auto Sprint, ZAim, and more.

## What It Does

- adds in-game toggle combos for multiple BO3 mods
- uses rumble and lightbar feedback for state changes
- supports Zombies and Multiplayer behavior differences
- exposes script-side timing and recoil values for manual tuning

## Basic Usage

1. Load the `.gpc` file into Cronus Zen Studio.
2. Put it into a slot on the Zen.
3. Make sure your in-game button layout matches the script's expected layout.
4. Use the toggle combos from the full guide linked above.

## Example Toggles

Rapid Fire:

```text
ADS + LEFT
```

Jitter mode cycle:

```text
ADS + UP
```

Drop Shot:

```text
ADS + DOWN
```

## Good To Know

- This is not a normal command-line script, so there are no CLI arguments.
